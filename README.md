# 💼 Portfólio Pessoal — Jhonatan Fernandes Santana

Este projeto consiste no desenvolvimento de um site de portfólio pessoal com o objetivo de apresentar informações profissionais, formação acadêmica, habilidades técnicas e projetos desenvolvidos.

O site foi construído utilizando **HTML, CSS e JavaScript puro**, sem uso de frameworks, com foco em organização estrutural, responsividade e interatividade.

---

## 📌 Objetivo

O principal objetivo deste projeto foi:

- Desenvolver um site de portfólio funcional e responsivo
- Aplicar conceitos de HTML semântico, CSS Grid/Flexbox e JavaScript
- Criar navegação dinâmica entre seções (modelo SPA)
- Implementar alternância de tema claro/escuro
- Estruturar conteúdo profissional para uso real no mercado

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts

---

## 🎯 Funcionalidades

- Navegação dinâmica entre seções (sem recarregamento de página)
- Alternância de tema Dark / Light
- Layout responsivo para diferentes tamanhos de tela
- Seção de formação com navegação interna
- Seção de portfólio com projetos e links externos
- Seção de contato com links diretos para redes e formulário acadêmico
